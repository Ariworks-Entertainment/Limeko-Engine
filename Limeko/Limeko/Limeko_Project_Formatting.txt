Limeko Project Formatting
(Concept)


#= Creation Flow =#

Create(Button) ->
  - Project Name
  - Project Developer
  - Project Publisher (Optional)
  - RP-Preset (Default: Citron)
  - Create (Button // Confirm)

Internal system creates Project folders.


#= Project Formatting =#

Parent folders with "(*)" are included in the project-
-explorer, and can be seen from within the editor.
Parent folders without it cannot be seen in-editor.

ProjectName
  ↳ Assets (*)
    ↳ Models (Empty)
    ↳ Materials (Empty)
    ↳ Resources (Empty)
    ↳ Levels (DefaultLevel.level)
  ↳ Settings (*)
    ↳ RenderSettings.json
  ↳ Editor (Empty)
